## 🚀 核心功能实现

### 1. AI策略解析模块

**文件**: `strategies/ai_parser.py`

**功能**:
- 使用Ollama本地模型将自然语言描述的策略转换为可执行的Python代码
- 支持多种AI模型（Qwen2-7B, Llama3-8B, Gemma2-9B）
- 自动生成策略规则列表和完整的策略代码
- 支持不同策略类型（量能策略、价格策略、混合策略、自定义策略）

**关键特性**:
```python
# 核心接口
def parse_strategy(strategy_desc: str, model: str = "ollama:qwen2:7b", **kwargs) -> Dict[str, Any]:
    """将自然语言策略描述解析为结构化规则和代码"""
```

**工作流程**:
1. 用户输入自然语言策略描述
2. 系统调用AI模型解析策略逻辑
3. 生成结构化的策略规则列表
4. 自动生成可执行的Python策略代码
5. 支持策略保存和回测

### 2. 选股漏斗页面

**文件**: `ui.py`

**功能**:
- 集成filter_engine，实现真实数据筛选
- 提供漏斗统计和可视化
- 实时显示选股结果表格
- 支持加载动画和扫描按钮

**关键特性**:
- 多维度股票筛选（技术指标、基本面、资金流向等）
- 智能评分系统，综合评估股票潜力
- 支持自定义筛选参数
- 诊股按钮快速查看个股分析

### 3. 策略实验室页面

**文件**: `ui.py`

**功能**:
- 自然语言策略输入
- AI模型选择
- 策略类型分类
- 多标签结果展示（规则列表、代码视图、回测结果）
- 策略保存和回测触发

**关键特性**:
- 异步处理，保持UI响应性
- 详细的策略规则展示
- 可编辑的策略代码视图
- 实时回测结果展示

## 🛠️ 技术架构

### 系统架构
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│    Web UI       │    │   策略引擎      │    │   数据引擎      │
│  (NiceGUI)      │───▶│  (Strategy)     │───▶│  (SQLite)       │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                        │                        │
         ▼                        ▼                        ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   AI解析模块    │    │   回测引擎      │    │   数据更新      │
│  (Ollama)       │    │  (Backtest)     │    │  (Scheduler)    │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 核心技术栈
- **前端框架**: NiceGUI (Python Web UI框架)
- **后端语言**: Python 3.13+
- **数据库**: SQLite (股票数据存储)
- **AI模型**: Ollama本地运行（Qwen2-7B, Llama3-8B等）
- **数据可视化**: Pyecharts
- **模板引擎**: Jinja2
- **任务调度**: 线程池异步执行

## 📝 使用说明

### 1. 启动系统
```bash
cd d:\StockAI
python ui.py
```

### 2. 访问界面
打开浏览器访问: `http://localhost:8080`

### 3. 使用AI策略解析
1. 点击左侧菜单的"策略实验室"
2. 输入自然语言策略描述
3. 选择AI模型和策略类型
4. 点击"开始解析"
5. 查看生成的规则和代码
6. 保存策略或运行回测

### 4. 使用选股漏斗
1. 点击左侧菜单的"选股漏斗"
2. 配置筛选参数
3. 点击"开始扫描"
4. 查看筛选结果
5. 点击"诊股"查看个股分析
